A Quick Guide


Use the Sixtus Cloud Scanner to become compliant with GDPR and help raise awareness of the facets of unstructured data

1. Overview and Data insight

Use the Overview Page to get data insight and filter for a more narrow and specific data set of the account(s) relevant to the query.

View checkpoints to see GDPR observation progression for comparsion between current status and previous GDPR case counts

When data has been found, click on the report-button to access the data in row of cases.

2. Examine GDPR Observations

Examine GDPR cases found in emails or files and unfold to see the textual components of each case and their GDPR classification and categorisation

If you wish to find different data, simply use the filter-button in the buttom corner

If GDPR cases prove irrelevant, mark the mail(s) in the square to the left and transfer the mail to "GDPR Quarantine" folder in Outlook. Hold Shift to mark multiple entries.
Note: The Folder is createad upon the first deletion of a mail, and should not be created manually.

Be aware that only mails are deletable via. the delete button.

The statistics on the Overview page will shortly update following any deletion.

3. Download CSV-file of Progression

Whereever a green CSV-file button is located, users are able to download the set of data in view.

If a user selects an Area on the overview page, only the data of that area will be visible in the CSV. If no data filtering has been done on the Overview page, all data available is selected by default.

4. Superusers & Configuration

Superusers have a third banner on the left hand side where they are able to perform Tenant-wide actions.

Remember to add users to be scanned or select which users to be skipped before the scanner is running - it selects none by default.

Superusers may also program the company policy for automated Quarantine of specific GDPR-classifications and set timers individual timers per Area
